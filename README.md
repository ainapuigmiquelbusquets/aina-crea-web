# aina-crea-web
Aina Crea neix de la passió per crear amb les mans i transformar idees en peces úniques. Un projecte artesanal on la creativitat i petits detalls són protagonistes.  Cada creació és feta a mà i personalitzada, combinant tècniques, materials i estils per donar vida a peces amb ànima, pensades per regalar, decorar o fer especial qualsevol moment.
